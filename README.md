Rules to follow when working with this repository.
1. Always work by forking the repository
2. Always create a new branch when working
3. The commit message should be descriptive about the changes you made in the code, like, "Create form for login", "Implement logic for user authentication", etc. And don't write the commit like "Made changes in the code", "Created some files", etc.
4. Always push the code to the development branch, not the main branch.
5. Always inform the owner about the changes you made.
